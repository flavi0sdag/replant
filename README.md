# Welcome to your Lovable project

## Project info

**URL**: https://replant-app.netlify.app/

### About The Project

RePlant is a platform designed to **gamify recycling and promote sustainable habits** through rewards, impact tracking, and community engagement.

The system allows users to register recyclable materials through **QR codes**, earn points, and visualize their environmental impact in real time. In addition, the platform connects recyclable materials with **local artisans**, encouraging a circular economy where waste becomes resources for new products.

The goal of this project is to make recycling:

- Simple
- Engaging
- Measurable
- Community-driven

### Core Features

- QR code recyclable registration
- Points and rewards system
- Gamification (missions, badges, ranking)
- Recycling traceability
- Local artisan product catalog
- Recycling drop-off point locator
- Environmental impact dashboard

## What technologies are used for this project?
